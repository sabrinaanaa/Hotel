Votre santé est notre Priorité

 

Dans le respect des mlesures préventives « anti- COVID-19», nous aimerions vous informer sur les points suivants:

- Gardez une distance d’ 1 mètre

- Lavez vos les mains le plus souvent possible

- Le gel désinfectant est disponible dans tous les locaux

- Portez un masque dans les espaces publics       

- Evitez le contact, et réduisez le plus possible de contact (en cas de besoin prière d’utiliser le téléphone)

- L'utilisation de l'ascenseur est autorisée pour 2 personnes uniquement dans le respect de la distanciation physique.

- Si vous vous sentez malade, contactez directement la réception

- Respecter le règlement de la piscine.       

- Selon le protocole, votre chambre sera nettoyée et désinfectée quotidiennement: Favoriser l’aération de la chambre  le plus possible. Laisser la porte du balcon ouvertes le jour de votre départ. 

- Check-out: Si la facture doit être payée, faites-le avant votre départ pour éviter la circulation à la réception.

- Respectez l'heure de départ (midi) 12H00
